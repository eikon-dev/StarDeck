# StarDeck Glossary

## Engine
Система, инкапсулирующая бизнес-логику и управляющая процессами на основе состояния.
Примеры: FXRunner, RewardEngine.

## Store
Источник истины состояния приложения.
Примеры: TaskStore, StarsStore, FXStore.

## Player
Исполнитель эффекта или процесса во времени.
Управляет жизненным циклом и обновляется на каждом кадре.
Примеры: StarPlayer.

## Port
Граница (boundary) между слоями системы.
Определяет контракт взаимодействия без знания реализации.
Примеры: FXPort, FXScenePort.
